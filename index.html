> Deployment

1. Deploy to Vercel: Connect your Git repository containing these files (`index.html`, `server.js`, `package.json`, `vercel.json`) to Vercel.

2. Set Env Variables: In your Vercel project settings, add two environment variables:

  • `SQUARE_ACCESS_TOKEN`: Your secret Square token.
  • `DEV_TOKEN`: The 6-digit token you want to use for access.

3. Use the Tool: Access your Vercel deployment URL. You will be prompted for the developer token to begin.
